# Django Mastery

This repository is my journey to mastering **Django**, from basic projects to advanced, production-ready applications.

---

## Skills & Features

- Build modular Django projects with multiple apps
- Custom user models, registration, login, logout
- Database management (SQLite, PostgreSQL, MySQL)
- Handle static files (CSS, JS, images) & integrate Bootstrap
- Upload and manage media files
- Apply permissions & role-based access
- Search & filtering functionalities
- Write tests for apps and features
- Advanced authentication (email-only login, social login via `django-allauth`)
- Configure environment variables for secure settings
- Performance optimizations (debug toolbar, indexing, caching)
- Apply security best practices for production
- Deploy projects with proper static handling & `ALLOWED_HOSTS`

---

## Sample Projects

- Online Bookstore: books, reviews, search, media uploads
- Pages App: home, about, contact
- E-commerce style features: filters, permissions, user roles
